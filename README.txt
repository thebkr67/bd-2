Что умеет бот:
- отвечает только в комментариях к постам канала
- отвечает на любые комментарии, кроме тех, где есть фраза "в личку"
- читает лимит из поста по шаблону: "МЕСТ 2"
- пишет "Место 1/2 занято", "Место 2/2 занято"
- не даёт одному и тому же пользователю занять 2 места под одним постом
- после заполнения пишет "Набор закрыт"

Пример:
В посте есть строка:
МЕСТ 2

Тогда:
- первый подходящий комментарий получает ответ "Место 1/2 занято"
- второй подходящий комментарий получает ответ "Место 2/2 занято"
- после этого бот пишет "Набор закрыт"
- следующие комментарии получают ответ "Набор закрыт"

Игнорируются комментарии:
- "в личку"
- "написал в личку"
- любые другие, где встречается фраза "в личку"

Важно:
- бот хранит данные в файле bot_state.sqlite3
- если удалить этот файл, счётчики обнулятся
- бот должен быть добавлен именно в группу обсуждения канала
- privacy mode у бота должен быть выключен
