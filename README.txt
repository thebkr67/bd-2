Telegram бот: автоответ на '+' в комментариях канала

Что делает бот:
- Отвечает 'принято' на комментарий '+'
- Работает только в комментариях к постам канала

Настройка:

1. Создайте бота через @BotFather
2. Скопируйте TOKEN
3. В файле bot.py вставьте токен:

BOT_TOKEN = "ВАШ_ТОКЕН"

4. Отключите privacy mode:
/setprivacy → Disable

5. Добавьте бота в группу обсуждения канала (лучше админом)

Установка:

pip install -r requirements.txt

Запуск:

python bot.py
